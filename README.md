# Setup

```
$ yarn create react-app cat-tinder-frontend
$ cd cat-tinder-frontend
```

Add the remote from your GitHub classroom repository

```
$ git checkout -b main
$ git add .
$ git commit -m "Initial commit."
$ git push origin main
$ yarn start
```
